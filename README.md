# Skillroom

Taproom-level terminal UI for managing Agent Skills.

## Vision

Skillroom is planned as a local-first, keyboard-first TUI for discovering,
inspecting, installing, updating, and removing Agent Skills across local and
marketplace sources.

## Current Stage

Current milestone: M0, repository and engineering baseline.

Run the minimal CLI:

```bash
cargo run
```

```bash
cargo check
```
